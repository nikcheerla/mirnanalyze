The Cancer Genome Atlas (TCGA), a project of the National Cancer Institute and the National Human Genome Research Institute (NHGRI), is the foundation of a large-scale collaborative effort to understand the genomic changes that occur in cancer. For more information, see http://cancergenome.nih.gov.

The files in this archive contain data derived from tissue samples of patients that were diagnosed with Cholangiocarcinoma. The data was produced by the Nationwide Children's Hospital (nationwidechildrens.org) Biospecimen Core Resource (BCR) using the Biospecimen Metadata - Complete Set platform.

This archive is the 13th revision of the 0th archive and contains only Level 2 (processed) data.
